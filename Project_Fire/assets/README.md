# 📁 Project Fire Assets

This folder is used for project-wide resources that are shared across the backend, frontend, and mobile apps.

### 🗄️ Suggested Folder Structure:
- **`logos/`**: Store your app's official logo and branding here.
- **`docs/`**: Store any additional PDF manuals or research papers here.
- **`screenshots/`**: Keep captures of the system in action for presentations.
- **`models/`**: (Already in `mobile_app/flutter_app/assets/models/`) Keep a master backup of your `.pt` or `.tflite` models here.

### 🔒 Privacy Note:
This folder is NOT ignored by `.gitignore` unless you add specific files/folders to it. Avoid putting sensitive credentials here!
